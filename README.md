# Cloud-Infrastructure-Automation-using-AWS-CloudFormation-Project-by-sairaj
##  Project Description
This project demonstrates Infrastructure as Code (IaC) using AWS CloudFormation.
The goal is to automate the provisioning of AWS resources and reduce manual setup time.


## Services Used
- AWS CloudFormation
- Amazon EC2
- Amazon S3
- Amazon VPC
- AWS IAM
- Git & GitHub

- Route Table
- EC2 Instance with IAM Role
- S3 Bucket

## How to Deploy
1. Login to AWS Console
2. Open CloudFormation
3. Create Stack -Upload template file
4. Select `template.yaml`
5. Create Stack
